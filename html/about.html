About

Artificial Life is an artificial life simulation project originally published on 314reactor.com. It uses a Raspberry Pi computer and a Unicorn Pi Hat to simulate artificial lifeforms. The following open source projects were used as part of this project:

  • WebIOPi (Thortex version)- https://github.com/thortex/rpi3-webiopi
  • Bare Bootstrap Template - https://startbootstrap.com/templates/bare/
  • Bootstrap - https://getbootstrap.com/
  • jQuery - https://jquery.com/
